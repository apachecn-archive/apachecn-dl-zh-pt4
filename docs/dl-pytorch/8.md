# 开放式神经网络交换

**开放神经网络交换**(**ONNX**)(【http://onnx.ai/】T4)是迈向开放生态系统的第一步，随着项目的发展，它将使你能够选择正确的工具。ONNX 为深度学习模型提供了开源格式。它定义了一个可扩展的计算图模型，以及内置运算符和标准数据类型的定义。Caffe2、PyTorch、微软认知工具包、Apache MXNet 等工具都在开发 ONNX 支持。这个项目可以帮助产品电离 PyTorch 模型。
